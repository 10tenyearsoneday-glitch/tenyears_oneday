# tenyears_oneday 模板（純前端示範）

這份模板是「可直接丟到 GitHub Pages / Netlify / 任何靜態空間」就能跑的版本。

## 功能
- 分頁導覽：關於我們、全系列、純銀飾品、優惠活動、飾品小知識、相關問題
- 右上角：IG / LINE / 搜尋 / 購物車 / 會員
- 商品列表 + 商品詳情（可多圖、可放大預覽、可選款式、數量加入購物車）
- 購物車：調整數量、刪除、結帳（7-11 / 全家 / 宅配），滿額免運、首購折扣、壽星折扣
- 會員：手機＋密碼登入/註冊；可查看訂單與修改資料（生日不可改）
- 後台：公告、運費/免運門檻可改；商品 CRUD；會員訂單數；訂單可刪除
- 資料儲存：全部存在瀏覽器 localStorage（示範用）

## 重要提醒（安全性）
這是「純前端」示範，所以：
- 後台帳密寫在前端檔案裡 **不安全**。
- 訂單/會員/商品都存在使用者瀏覽器（換裝置就沒了）。
正式營運請改成「後端 + 資料庫 + 金流」架構。

## 上線方式（GitHub Pages）
1. 建立 repo（例如：tenyears-site）
2. 把整包檔案上傳到 repo 根目錄
3. Settings → Pages → Source 選 main / root → Save
4. 等待 Pages 生效後開啟網址

## 後台入口（隱藏）
頁尾左下角有一個「完全隱形」的連結（可點）：
- admin/login.html
